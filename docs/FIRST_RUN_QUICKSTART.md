# First Run Quickstart

If this is your first time in Adam, do this:

1. Run `.venv/bin/python -m eden`.
2. Wait for the live chat screen.
3. Type in `Message Input`.
4. Press `Ctrl+S`.
5. Read Adam's reply in `Adam Dialogue`.
6. Press `Esc` if you want to go straight back to typing.

Useful next actions:

- `F9` loads a document before the first turn.
- `F6` opens the detailed deck.
- `F8` opens aperture.
- `F10` opens the saved-conversation archive.
- `F2` exports the current experiment's browser artifacts.
- `F3` opens the browser observatory.

If the terminal is small:

- Adam now switches into a compact layout automatically.
- The right telemetry stack is hidden by default so the transcript and composer stay usable.
- `F8` becomes a temporary aperture view and `Esc` returns to dialogue.

If focus feels wrong:

- Press `Esc`.
